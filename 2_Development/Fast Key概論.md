# 功能區塊
- 未知功能先往那個方向找?
  - shift 反向含義
  - Alt 變更含義
    - ex:Visual Studio ctrl+shift+f3時 alt+a 就會有click button效果
    - ex:excel copy sheet時,alt+c select checkbox
  - F6、tab跳躍可先試
  - ctrl,win+num 快速進到某個應用程式次序
  - 每個AP都會有預設focus
- 增加效率技巧?
  - 應用程式固定在指定位置
  - 應用程式區塊切換記得要移動幾次
